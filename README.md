# 📂 Featured Projects

## 🔹 HR Employee Attrition Analysis

🔗 Project Link:  
https://github.com/poojitha0101/hr-attrition-analysis

### 📌 Business Problem
A company was facing a high employee attrition rate of 16.12%.  
This project analyzes employee behavior and identifies key factors driving attrition.

### 📊 Key Insights
- Identified overall attrition rate of **16.12%** (237 employees)
- Sales department showed highest attrition rate (**20%**)
- Employees working overtime were **3x more likely** to leave
- Employees who left earned significantly lower average salaries
- Age group **26–35** recorded highest attrition count

### 🛠 Tools Used
Python · Pandas · Matplotlib · Seaborn · SQL Server · Power BI · Excel
